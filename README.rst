Sparse accumulation
===================

Optimized pytorch CUDA and cpu kernels for the sparse accumulation operation which is usefull for SO(3) equivariant computations

++++++++++++
Installation
++++++++++++

:bash:`python3 -m pip install .`

++++++++++++
Tests
++++++++++++

:bash:`python3 -m pytest test_cpp_jit_cuda.py -vrA`
